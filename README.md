## About FAIRGO

Fairgo is a code-as-law blockchain game that optimizes pareto.see [https://fairgogame.github.io](https://fairgogame.github.io/).

## Rule 

* 42% static incomes  
* 30% Referral bonuses
* 12% Shareholder dividend income
* 10% Bonus pools
* 3%  Community revenue
* 3%  The technical team
